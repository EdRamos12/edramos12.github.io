O propósito deste aplicativo, é de conectar alunos e professores, por meio de uma interface em rede social, para tornar a navegação mais eficiente. A plataforma é feita para instrutores postarem aulas que estão disponíveis dar aula, e ao estudante entrar no aplicativo, ele poderá procurar por matérias específicas, datas disponíveis do professor, e ainda filtrar pelo valor da hora/aula

O desenvolvimento deste projeto foi feito por fins estudantis, parte do servidor e páginas web foram feitas inteiramente por mim com React e NodeJS.

Golden Resource

Projeto Pessoal

JavaScript HTML5 CSS3 Git Electron

Aplicativo desktop, feito para gravar tela do usuário e também fazer isso com janelas específicas. Logo após o término da gravação, o app "renderiza" o vídeo produzido pelo browser, para que saia um arquivo .mp4 funcional, convertido pela library FFMpeg.

Software foi feito baseado em Eletron, FFMpeg, JSON (para configurações), e incluso junto, um auto-updater para quando novas versões forem lançadas, fazer o download automaticamente ao iniciar o app.

Aplicativo feito para fins estudantis, design e parte do código foram inteiramente feitos por mim.
